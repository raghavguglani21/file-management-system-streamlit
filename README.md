# 📁 Smart File Management System

A modern File Management System built with Python and Streamlit.

## Features

* Create Files
* Read Files
* Append Content
* Overwrite Files
* Rename Files
* Delete Files
* Modern Dashboard UI
* User-Friendly Interface

## Tech Stack

* Python
* Streamlit
* Pathlib
* OS Module

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python -m streamlit run app.py
```
